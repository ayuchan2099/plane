# ✈️ 飞机大战 - 惊险刺激版

一款画面鲜艳、场面惊险、玩法刺激的飞机大战游戏！

## 🎮 游戏特色

### 场面惊险
- **多种敌机类型**：小型、中型、大型敌机和超强BOSS
- **弹幕系统**：密集的子弹和多样化的攻击模式
- **碰撞检测**：精准的碰撞系统，每次躲避都惊险刺激
- **难度递增**：游戏随着分数增加而变得更具挑战性

### 画面鲜艳
- **霓虹色彩**：使用青色、紫色、黄色等鲜艳的霓虹配色
- **粒子特效**：爆炸、尾迹、星空等丰富的粒子效果
- **光影效果**：子弹和飞机都带有发光效果
- **流畅动画**：60FPS流畅运行，动画过渡自然

### 玩法刺激
- **5级武器系统**：从单发到激光，武器不断升级
- **道具收集**：武器升级、生命恢复、护盾、炸弹等多种道具
- **连击系统**：连续击杀可触发连击奖励
- **炸弹系统**：危急时刻可使用炸弹清屏
- **BOSS战**：达到分数后挑战强大BOSS

## 🎯 操作说明

### 电脑端
- **鼠标移动**：控制飞机移动
- **空格键**：使用炸弹（清除屏幕上所有敌人和子弹）
- **自动射击**：飞机会自动发射子弹

### 移动端
- **触摸移动**：手指在屏幕上移动控制飞机
- **自动射击**：飞机会自动发射子弹

## 🎖️ 游戏系统

### 动态难度系统 ⭐ 新增
- **难度等级**：每200分提升一个难度等级
- **敌机种类**：随难度解锁新敌机类型
- **生成频率**：难度越高，敌机出现越频繁
- **同时数量**：高难度下最多同时出现4架敌机
- **道具平衡**：
  - 掉落率：20% → 5%（随难度递减）
  - 生成间隔：15秒 → 30秒（随难度增加）
  - 高难度更多生命和护盾道具
- **动态背景** ⭐ 新增：
  - 背景渐变色随难度变化（蓝→紫→红）
  - 星星颜色随难度改变
  - Lv.3+ 出现星云效果
  - Lv.4+ 出现流星
  - Lv.5+ 星星密度增加
  - Lv.7+ 进入危险红色区域
- **视觉提示**：右上角实时显示当前难度等级和道具掉落率

### 敌机类型（8种）
1. **小型敌机** (10分) - 基础
   - 生命值：1 | 速度：2.5
   - 移动模式：直线俯冲
   - 解锁难度：Lv.1

2. **中型敌机** (30分) - 基础
   - 生命值：3 | 速度：2.0
   - 移动模式：之字形摆动
   - 解锁难度：Lv.1

3. **大型敌机** (100分) - 基础
   - 生命值：8 | 速度：1.3
   - 移动模式：圆周运动
   - 解锁难度：Lv.1

4. **快速小型机** (20分) ⭐ 新增
   - 生命值：1 | 速度：4.0
   - 移动模式：快速冲刺
   - 解锁难度：Lv.3

5. **重型机** (150分) ⭐ 新增
   - 生命值：15 | 速度：0.8
   - 移动模式：大幅波浪
   - 解锁难度：Lv.3

6. **精英机** (120分) ⭐ 新增
   - 生命值：10 | 速度：1.8
   - 移动模式：螺旋下降
   - 解锁难度：Lv.5

7. **狂暴机** (80分) ⭐ 新增
   - 生命值：6 | 速度：2.5
   - 移动模式：不规则激进
   - 解锁难度：Lv.7

8. **BOSS** (500分)
   - 生命值：60 | 速度：0.7
   - 移动模式：复杂路径
   - 攻击模式：密集扇形弹幕
   - 每400分出现一次

### 道具系统
- **🔥 武器升级**：提升武器等级（最高5级）
- **❤️ 生命恢复**：恢复30点生命值
- **🛡️ 护盾**：5秒内免疫伤害
- **💣 炸弹**：增加一个炸弹（最多5个）

### 炸弹系统限制 ⭐ 新增
- **最大数量**：最多携带5个炸弹
- **冷却时间**：使用后需要20秒才能再次使用
- **冷却显示**：右下角显示冷却进度条和倒计时
- **满弹补偿**：炸弹已满时拾取炸弹道具会获得50金币
- **视觉反馈**：使用炸弹时显示击杀数量
- **失败提示**：冷却中或无炸弹时会显示提示信息

### 武器等级
- **等级1**：单发子弹
- **等级2**：双发子弹
- **等级3**：三发子弹
- **等级4**：扩散子弹（三向射击）
- **等级5**：激光+扩散（终极火力）

### 微信小游戏适配

## 🌟 游戏技巧

1. **保持移动**：不要停在一个地方，持续移动可以躲避更多子弹
2. **收集道具**：优先收集武器升级道具，提升火力
3. **炸弹策略** ⭐ 重要：
   - 炸弹有20秒冷却，不要浪费
   - 最多携带5个，合理规划使用时机
   - 在被包围或BOSS战时使用
   - 冷却中时要更加谨慎走位
   - 注意右下角的冷却进度条
4. **连击奖励**：尽量连续击杀敌机获得连击
5. **护盾利用**：获得护盾后可以大胆冲进敌群
6. **高难度生存**：难度越高道具越少，要珍惜每个道具
